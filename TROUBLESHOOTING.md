## 🔐 בעיות כניסה והרשמה

### ❌ "Invalid login credentials"

**סיבות אפשריות:**
1. סיסמה שגויה
2. אימייל לא רשום במערכת
3. שגיאת הקלדה באימייל

**פתרון:**
```
✅ שלב 1: בדקו שהאימייל נכתב נכון
   - אין רווחים מיותרים
   - אותיות גדולות/קטנות לא משנות

✅ שלב 2: נסו "איפוס סיסמה" (אם יש)
   - או צרו חשבון חדש

✅ שלב 3: בדקו שההרשמה הושלמה
   - ייתכן שהרישום לא הסתיים
```

---

### ❌ "Email already registered"

**משמעות:** האימייל כבר קיים במערכת

**פתרון:**
```
✅ האם שכחתם שנרשמתם?
   → נסו להתחבר במקום להירשם

✅ רוצים חשבון נוסף?
   → השתמשו באימייל אחר

✅ אולי מישהו אחר השתמש באימייל?
   → פנו למנהל המערכת
```

---

### ❌ "Password must be at least 6 characters"

**פתרון:**
```
✅ הסיסמה חייבת להכיל 6 תווים לפחות
   דוגמה נכונה: "abc123"
   דוגמה שגויה: "abc12" (רק 5)
```

---

### ❌ "Session expired" או "User not logged in"

**משמעות:** ה-Session פג תוקף

**פתרון:**
```
✅ התנתקו והתחברו מחדש
   לחצו על שם המשתמש → "התנתק"
   
✅ אם זה קורה הרבה:
   - נקו cookies של הדפדפן
   - נסו דפדפן אחר
```

## 📝 בעיות יצירת תכנים

### ❌ "אין מספיק קרדיטים"

**משמעות:** אין מספיק קרדיטים לפעולה

**פתרון:**
```
✅ בדקו כמה קרדיטים יש לכם:
   ראו את המספר ליד שם המשתמש (למעלה)

✅ אם נגמרו:
   פנו למנהל המערכת להוספת קרדיטים

✅ חסכו בקרדיטים:
   - השתמשו ב-"רענן" (חינם) במקום ליצור חדש
   - תכננו מראש את מספר החלקים
```

**דוגמה:**
```
המצב:
💎 יש לך: 1 קרדיט
📝 הפעולה עולה: 2 קרדיטים
❌ תוצאה: "אין מספיק קרדיטים"

הפתרון:
פנו למנהל → הוסיפו עוד 10 קרדיטים
עכשיו: 11 קרדיטים ✅
```

---

### ❌ "נא לבחור תחום ידע ונושא"

**משמעות:** שכחתם למלא שדה חובה

**פתרון:**
```
✅ במרכז הלמידה:
   1. בחרו תחום ידע (מתמטיקה, מדעים...)
   2. הזינו נושא או בחרו מההצעות
   3. רק אז לחצו "צור"

✅ ודאו שהשדות לא ריקים!
```

---

### ❌ "API did not return text"

**משמעות:** בעיה בחיבור ל-AI

**סיבות אפשריות:**
1. בעיית רשת
2. שרתי Google עמוסים
3. תוכן לא מתאים (נדיר)

**פתרון:**
```
✅ המתינו 30 שניות ונסו שוב

✅ רעננו את הדף (F5)

✅ בדקו חיבור לאינטרנט

✅ אם נמשך:
   - נסו נושא אחר
   - פנו למנהל
```

---

### ❌ הכפתור "יצור" מושבת (אפור)

**סיבות אפשריות:**

**1. אין קרדיטים:**
```
בדקו: 💎 מספר הקרדיטים בסרגל העליון
פתרון: פנו למנהל להוספת קרדיטים
```

**2. שדות חובה ריקים:**
```
בדקו:
- במרכז למידה: תחום + נושא מלאים?
- בסיפורים: רעיון הוזן?
פתרון: מלאו את כל השדות הנדרשים
```

**3. אין פרופיל פעיל:**
```
בדקו: האם יש פרופיל של ילד?
פתרון: צרו פרופיל ראשון → "הגדרות פרופיל"
```

**4. המערכת עסוקה:**
```
בדקו: האם יש Loader (אנימציה) על המסך?
פתרון: המתינו עד שהטעינה תסתיים
```

---

### ❌ התמונה לא נטענת / תמונה שחורה

**סיבות אפשריות:**
1. תמונה גדולה מדי
2. פורמט לא נתמך
3. בעיה ב-AI

**פתרון:**
```
✅ לחצו "🔄 רענן" (חינם!) - יצור תמונה חדשה

✅ אם זה חוזר:
   המשיכו בלי תמונה - הטקסט יישאר

✅ בפרופיל:
   - השתמשו בתמונות JPG/PNG
   - גודל מומלץ: מתחת ל-2MB
   - רזולוציה: 500x500 - 1000x1000
```

---

### ❌ "המערכת לא מגיבה" / תקוע באמצע יצירה

**פתרון מהיר:**
```
🔴 אל תסגרו את הדף!

✅ המתינו עד דקה - ייתכן שהשרת עמוס

✅ אם עברה דקה:
   1. רעננו את הדף (F5)
   2. היכנסו לספרייה
   3. בדקו אם נשמר חלק מהתוכן

✅ אם לא נשמר:
   צרו מחדש (התוכן לא חויב בקרדיטים אם נכשל)
```

```
✅ המתינו 10 שניות - עדכון אוטומטי

✅ אם לא עובד:
   התנתקו והתחברו מחדש
   
✅ כפתור חירום:
   רעננו את הדף (F5)
```

---

### ❌ "חויבתי קרדיטים אבל התוכן לא נוצר"

**מה קרה?**
```
אם היצירה נכשלה לפני השמירה:
→ הקרדיטים חויבו
→ התוכן לא נשמר

זה נדיר אבל יכול לקרות
```

**פתרון:**
```
✅ צרו screenshot של השגיאה

✅ פנו למנהל עם:
   - זמן הבעיה
   - מה ניסיתם ליצור
   - Screenshot
   
✅ המנהל יוכל להחזיר קרדיטים ידנית
```

---

### ❌ "עלות הקרדיטים השתנתה באמצע יצירה"

**משמעות:** מנהל-על שינה מחירים

**פתרון:**
```
✅ זה תקין - מחירים דינמיים

✅ אתם משלמים לפי המחיר שהיה כשהתחלתם

✅ יצירות חדשות ישתמשו במחיר החדש

✅ אם אין לכם קרדיטים למחיר החדש:
   פנו למנהל להוספת קרדיטים
```

## 👤 בעיות פרופילים

### ❌ "אין לך פרופילים פעילים"

**משמעות:** חייבים ליצור פרופיל של ילד

**פתרון:**
```
✅ לחצו "הגדרות פרופיל" (בסרגל העליון)

✅ "+ הוסף פרופיל"

✅ מלאו:
   שם: [שם הילד]
   גיל: [גיל]
   מין: [בן/בת]
   תחומי עניין: [עניינים]
   
✅ "שמור פרופיל"

עכשיו אפשר ליצור תכנים! ✅
```

---

### ❌ "לא רואה את הפרופיל החדש שיצרתי"

**פתרון:**
```
✅ בדקו שלחצתם "שמור פרופיל"

✅ רעננו את הדף (F5)

✅ בדקו בתפריט הפרופילים (למעלה)
   לחצו על שם הפרופיל → רשימת כל הפרופילים
```

---

### ❌ "התמונה שהעליתי לא מופיעה בפרופיל"

**סיבות:**
1. הקובץ גדול מדי (מעל 5MB)
2. פורמט לא נתמך
3. שגיאת העלאה

**פתרון:**
```
✅ גודל תמונה מומלץ: 500KB - 2MB

✅ פורמטים נתמכים: JPG, PNG, WEBP

✅ איך להקטין תמונה:
   - Windows: פתחו ב-Paint → Resize → Save
   - Mac: Preview → Tools → Adjust Size
   - אונליין: TinyPNG.com
   
✅ נסו להעלות שוב
```

---

### ❌ "מחקתי פרופיל בטעות!"

**⚠️ לצערנו - אין דרך לשחזר**

```
❌ מחיקת פרופיל היא קבועה
❌ כל התכנים של הפרופיל נמחקים

מה אפשר לעשות:
✅ צרו את הפרופיל מחדש
✅ התכנים ישנים אבודים
✅ בפעם הבאה: זהירות לפני מחיקה!
```

## 🐌 בעיות ביצועים וטעינה

### ❌ "המערכת איטית / תקועה"

**פתרונות לפי חומרה:**

```
✅ בדיקה ראשונה: חיבור לאינטרנט
   speedtest.net → ודאו שיש חיבור יציב

✅ רענון דף: F5

✅ ניקוי Cache:
   Chrome: Ctrl+Shift+Delete
   Firefox: Ctrl+Shift+Delete
   → סמנו "Cached images and files"
   → Clear data

✅ סגירת טאבים מיותרים
   המערכת צריכה זיכרון - סגרו טאבים שלא בשימוש

✅ נסו דפדפן אחר
   Chrome / Firefox / Edge - הכי מומלצים
```

---

### ❌ "הטעינה לוקחת יותר מ-2 דקות"

**מה תקין:**
```
⏱️ סיפור: 30-60 שניות
⏱️ תכנית למידה: 45-90 שניות
⏱️ חוברת: 60-120 שניות
```

**אם יותר:**
```
✅ בדקו מהירות אינטרנט (צריך לפחות 5 Mbps)

✅ נסו שוב - שרתי AI עשויים להיות עמוסים

✅ אם חוזר על עצמו:
   - נסו בשעות אחרות (פחות עומס)
   - פנו למנהל
```

---

### ❌ "הדף לא עולה / מסך לבן"

**פתרון:**
```
✅ שלב 1: Hard Refresh
   Windows: Ctrl+F5
   Mac: Cmd+Shift+R

✅ שלב 2: נקו Cache (ראו למעלה)

✅ שלב 3: בדקו Console
   F12 → Console → חפשו שגיאות באדום
   צלמו screenshot ושלחו למנהל

✅ שלב 4: נסו דפדפן אחר
```

## 🖨️ בעיות הדפסה

### ❌ "ההדפסה לא נראית טוב"

**פתרון:**
```
✅ ודאו שבחרתם:
   - Orientation: Portrait (אנכי)
   - Paper size: A4
   - Margins: Default

✅ במקום "Print":
   בחרו "Save as PDF" → ואז הדפיסו את ה-PDF
   
✅ אם התמונות לא נראות:
   בחרו "Background graphics" ON
```

---

### ❌ "הדפסתי אבל חסרים חלקים"

**סיבה:** דפי רקע לא נבחרו

**פתרון:**
```
✅ בחלון ההדפסה:
   More settings → סמנו "Background graphics"
   
✅ או: שמרו כ-PDF ואז הדפיסו
```

---

### ❌ "לא מוצא את כפתור ההדפסה"

**לפי סוג תוכן:**

```
📝 דף תרגול:
   הספרייה שלי → לחצו על הדף → 🖨️ הדפס

✍️ סיפור:
   פתחו את הסיפור → Ctrl+P (הדפסת דף רגילה)

📖 חוברת:
   פתחו את החוברת → Ctrl+P
```

## 🚨 שגיאות מערכת

### ❌ "Network Error" / "Failed to fetch"

**משמעות:** בעיית חיבור

**פתרון:**
```
✅ בדקו חיבור אינטרנט

✅ נסו גישה לאתרים אחרים

✅ בדקו Firewall / Antivirus
   ודאו שהם לא חוסמים את האפליקציה

✅ נסו רשת אחרת (WiFi אחר / נתונים סלולריים)
```

---

### ❌ "Supabase connection failed"

**משמעות:** בעיה בחיבור ל-Database

**פתרון למשתמשים:**
```
✅ רעננו את הדף (F5)

✅ נסו שוב בעוד כמה דקות

✅ אם נמשך - פנו למנהל
```

**פתרון למנהלי מערכת:**
```
✅ בדקו Supabase Dashboard:
   - Project active?
   - No billing issues?
   
✅ בדקו .env:
   - VITE_SUPABASE_URL נכון?
   - VITE_SUPABASE_ANON_KEY נכון?
   
✅ בדקו Supabase Status:
   status.supabase.com
```

---

### ❌ "Google AI API Error"

**פתרון למשתמשים:**
```
✅ נסו שוב בעוד דקה

✅ אם חוזר - פנו למנהל
```

**פתרון למנהלי מערכת:**
```
✅ בדקו API Key:
   - .env → VITE_API_KEY קיים?
   - המפתח תקף?
   
✅ בדקו Quota:
   Google AI Studio → Usage
   ייתכן שעברתם את הלימיט היומי
   
✅ בדקו Console:
   F12 → Console → חפשו הודעת שגיאה מפורטת
```

## 🌐 בעיות דפדפן

### ❌ "האפליקציה לא נראית טוב במובייל"

**פתרון:**
```
✅ המערכת responsive - אמורה לעבוד

✅ אם לא:
   - נסו לרענן (משיכה מלמעלה)
   - נסו Landscape mode (אופקי)
   - זום: 100%

✅ דפדפנים מומלצים למובייל:
   - Chrome (Android)
   - Safari (iOS)
```

---

### ❌ "הכפתורים לא עובדים"

**סיבות אפשריות:**
1. JavaScript מושבת
2. הרחבות דפדפן חוסמות
3. Ad-blockers

**פתרון:**
```
✅ נסו Incognito/Private mode:
   Chrome: Ctrl+Shift+N
   Firefox: Ctrl+Shift+P
   
✅ אם עובד שם:
   השעו הרחבות אחת אחת וראו מי מפריע
   
✅ ודאו JavaScript מופעל:
   Settings → Site Settings → JavaScript: Allow
```

---

### ❌ "הגופנים נראים מוזר / לא עבריים"

**פתרון:**
```
✅ רעננו את הדף (F5)

✅ נקו Cache של הדפדפן

✅ ודאו שבמחשב יש תמיכה בעברית:
   Windows: Language settings → Hebrew
   Mac: System Preferences → Language
```

## 🆘 פניה לתמיכה

### מה לכלול בפנייה למנהל:

```
📧 פנייה מומלצה:

נושא: בעיה ביצירת סיפור
```
