# ⚡ מדריך התחלה מהירה - ChildApp2

זקוקים להתחיל מהר? הנה המינימום שצריך לדעת!

---

## 🎯 5 דקות - מההרשמה ליצירה ראשונה

### 1️⃣ הרשמה (30 שניות)

```
פתחו את האפליקציה → "הרשמה"
↓
מלאו:
📧 אימייל: your@email.com
🔒 סיסמה: ******
👤 שם: שמכם
↓
לחצו "הרשם" → אתם בפנים! ✅
```

---

### 2️⃣ יצירת פרופיל ראשון (1 דקה)

```
לחצו "הגדרות פרופיל" (בסרגל העליון)
↓
"+ הוסף פרופיל"
↓
מלאו:
שם: נועה
גיל: 7
מין: בת
תחומי עניין: חיות, ציור
↓
"שמור" → הפרופיל מוכן! ✅
```

---

### 3️⃣ יצירת סיפור ראשון (3 דקות)

```
לחצו "✍️ סיפורים" (בתפריט)
↓
כתבו רעיון:
"נועה מוצאת גור חתולים ביער"
↓
לחצו "התחל סיפור!"
↓
⏳ המתינו 30 שניות...
↓
📖 קראו את החלק הראשון!
↓
לחצו "המשך סיפור" עוד 2-3 פעמים
↓
"סיים סיפור" → הסיפור נשמר! ✅
```

---

## 📚 פעולות בסיסיות

### ✍️ יצירת סיפור

| מה עושים | איך |
|----------|-----|
| התחלה | סיפורים → הזינו רעיון → "התחל" |
| המשך רגיל | "המשך סיפור" |
| המשך מיוחד | כתבו מה יקרה → "המשך עם ההנחיה" |
| שינוי חלק | כפתור "🔄 רענן" (חינם!) |
| סיום | "סיים סיפור" |

**עלות:** 1 קרדיט לכל חלק

---

### 📚 מרכז למידה

#### תכנית למידה

```
מרכז למידה → "תכנית למידה"
↓
בחרו תחום (מתמטיקה, מדעים...)
↓
לחצו "💡 הצע נושאים" → בחרו מהרשימה
↓
כל שלב = 2 קרדיטים
5 שלבים סה"כ
```

#### דף תרגול

```
מרכז למידה → "דף תרגול"
↓
בחרו תחום + כתבו נושא
(דוגמה: "חיבור עד 20")
↓
"צור דף" → קבלו דף להדפסה
↓
עלות: 2 קרדיטים
```

---

### 📖 הספרייה שלי

```
לחצו "הספרייה שלי"
↓
ראו את כל מה שיצרתם:
- סיפורים → לחצו "👁️ צפה"
- דפים → לחצו "🖨️ הדפס"
- תכניות → לחצו "👁️ המשך"
```

## 💎 קרדיטים - המינימום

### כמה עולה כל דבר?

| פעולה | עלות (ברירת מחדל) |
|-------|-------------------|
| 📝 חלק סיפור | 1 קרדיט |
| 📚 שלב תכנית | 2 קרדיטים |
| 📄 דף תרגול | 2 קרדיטים |
| 📖 חוברת | 3 קרדיטים |
| 💡 הצעות נושאים | 1 קרדיט |

### איך לחסוך?

✅ **משתמשים ב-"רענן"** במקום ליצור חדש (חינם!)
✅ **תכננו מראש** - 4-6 חלקים מספיק לסיפור
✅ **בחרו מהצעות** במקום לכתוב נושא חדש

---

## 👨‍💼 מנהלים - פעולות מהירות

### הוספת משתמש

```
ניהול משתמשים → "➕ הוסף משתמש"
↓
מלאו:
שם: רחל לוי
אימייל: rachel@example.com
סיסמה: ******
קרדיטים: 50
↓
"צור משתמש" → מוכן!
```

### הוספת קרדיטים

```
ניהול משתמשים → לחצו על שם המשתמש
↓
הזינו כמות → "הוסף קרדיטים"
↓
עדכון מיידי! ✅
```

### שינוי עלויות (Super Admin)

```
בראש דף "ניהול משתמשים" יש:
"⚙️ שליטה על עלויות יצירות במערכת"
↓
שנו מספרים → "שמור שינויים"
↓
עדכון לכל המשתמשים תוך 10 שניות! ✅
```

## 🐛 פתרון בעיות מהיר

### "אין מספיק קרדיטים"
👉 פנו למנהל להוסיף קרדיטים

### "טעינה תקועה"
👉 רעננו את הדף (F5)

### "כפתור מושבת"
👉 בדקו:
   - ✅ יש קרדיטים?
   - ✅ מילאתם את כל השדות?
   - ✅ יש פרופיל פעיל?

### "הסיפור לא נשמר"
👉 אל דאגה! שמירה אוטומטית אחרי כל חלק
   אם סגרתם באמצע - בדקו בספרייה

### "לא רואה עדכון עלויות" (מנהל)
👉 המתינו 10 שניות או לחצו "🔄 רענן עלויות"

## 💡 טיפים מהירים

### לסיפורים

✅ **רעיון ראשון:**
```
"[שם הילד] + [תחום עניין] + [מקום]"
דוגמה: "נועה פוגשת דולפין בים"
```

✅ **אורך אידיאלי:**
4-6 חלקים (לא ארוך מדי!)

✅ **תנו לילד להחליט:**
שאלו: "מה תרצה שיקרה עכשיו?"

---

### למרכז למידה

✅ **בחירת נושא:**
```
❌ רע: "מתמטיקה"
✅ טוב: "חיבור עד 20"
✅ מצוין: "חיבור עד 20 עם בעיות מילוליות"
```

✅ **גילאים:**
- 5-6: נושאים בסיסיים (מספרים עד 10, אותיות)
- 7-8: נושאים מתקדמים (חיבור/חיסור, קריאה)
- 9+: נושאים מורכבים (כפל, מבנה משפט)

---

### לפרופילים

✅ **תחומי עניין מפורטים:**
```
❌ רע: "חיות"
✅ טוב: "חיות, ציור, ריקוד"
✅ מצוין: "חיות במיוחד חד-קרן, ציור בצבעי מים, ריקודים מודרניים"
```

✅ **תמונה:**
אם מוסיפים תמונה → הדמויות בסיפורים נראות כמו הילד!
