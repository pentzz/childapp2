## ⚙️ הפאנל המתקדם - מה זה כולל?

### טאב 1: 📄 סקשנים
```
תפקודים:
├── ➕ הוסף סקשן חדש
├── ⋮⋮ גרור לשינוי סדר (drag & drop)
├── 👁️ הסתר/הצג סקשן
├── ✏️ ערוך כותרת ותת-כותרת
└── 🗑️ מחק סקשן (לא סקשני מערכת)
```

**איך להוסיף סקשן:**
1. לחץ "➕ הוסף סקשן חדש"
2. סקשן חדש מתווסף לרשימה
3. ערוך את הכותרת והתת-כותרת
4. גרור אותו למקום הרצוי
5. השינויים נשמרים אוטומטית!

### טאב 2: 🍔 תפריט
```
תפקודים:
├── ➕ הוסף פריט תפריט
├── ✏️ ערוך תווית
├── 🔗 ערוך קישור
├── 🎨 הוסף אייקון (אימוג'י)
└── 🗑️ מחק פריט
```

**איך לערוך תפריט:**
1. לחץ על טאב "תפריט"
2. כל פריט בכרטיס נפרד
3. ערוך שדות: תווית, קישור, אייקון
4. השינויים נשמרים בלחיצה מחוץ לשדה

### טאב 3: ⚙️ הגדרות
```
קטגוריות:
├── 🌐 כללי (כותרת, אימייל)
├── 🎨 עיצוב (צבעים, לוגו)
└── 🔍 SEO (תיאור, meta tags)
```

**איך לשנות צבעים:**
1. לחץ על טאב "הגדרות"
2. מצא "צבע ראשי" / "צבע משני"
3. לחץ על הריבוע הצבעוני
4. בחר צבע חדש
5. השינוי נשמר מיידית!
